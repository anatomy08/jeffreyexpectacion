About Me

With 2.7 years of experience in Unity development, I have hands-on experience in integrating cloud services with AWS, optimizing mobile and WebGL builds, and managing assets using Unity Addressables and Asset Bundles.

I've worked extensively with API integration and migration, version control systems like GitHub and SourceTree, and front-end technologies (HTML, CSS, JavaScript) to ensure browser compatibility.
